PushPop - Help

PushPop is a puzzle game based on the behavior of a stack: you can add (push) and remove (pop) things from the top (head) of the stack, but you can't get at the middle (tail).

The game is setup with four puzzle stacks with four pieces in each and an empty solution stack.

The point of the game is to pop all of the pieces from the puzzle stacks -- just tap the piece you'd like to pop -- and push them onto the solution stack. However, you can only push a piece onto the solution stack if it matches either the shape/number or color of the piece at the head of the solution stack.

If you try to pop a piece off a game stack that doesn't match the current head of the solution stack, the piece you've selected will wiggle a bit to let you know. If you feel like you've made a mistake, you can always unwind by popping pieces back off of the solution stack or by opting to Start Over. Starting over won't reset the timer or the moves counter, but it will get the stacks back to their original state so you can try again.

If you really get stuck, you can ask for a hint.

Every puzzle has a solution and some puzzles have more than one. See how fast you can solve them, and try to use the least number of moves.

Initially, only the 'Easy' level is enabled, but you can unlock the next version by solving a puzzle quickly with few mistakes. The 'Harder' and 'Insane' levels are only available in the paid version of PushPop.

Enjoy.
